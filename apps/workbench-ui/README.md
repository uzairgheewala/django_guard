# PlanGuard Workbench UI — Milestone B

The React/Vite UI renders canonical artifacts served by the Django workbench API.

```bash
npm install
npm run dev
```

Set `VITE_PLANGUARD_API_BASE` when the API is not at `http://127.0.0.1:8000`.

Milestone B surfaces:

- captured run registry;
- query execution timeline;
- family-lens switching;
- evidence-backed finding cards;
- detector receipts and persisted policy evaluations;
- Policy Studio for absolute query budgets;
- the Milestone A Artifact Inspector and capability registry.
